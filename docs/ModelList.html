items()

Source:
  • models/index.js, line 320

The set of items in the ModelList.

NOTE: Returns a shallow copy of the underlying collection. That means the array returned can be mutated without affecting the real ModelList, but all the items in the array are the same by reference, so mutating an object in the collection will also mutate the object stored in the ModelList.

Example
import { modelList, withOrdering } from '@paychex/core/models';

let list = modelList(1, 2, 3);
list.add(4, 5, 6);
console.log(list.items()); // [1, 2, 3, 4, 5, 6]
list = withOrdering(list, [], ['desc']);
console.log(list.items()); // [6, 5, 4, 3, 2, 1]
